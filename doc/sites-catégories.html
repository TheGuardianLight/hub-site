Sites & catégories

Pour ajouter un site et une catégorie, il faudra suivre chaque étape et éviter de cour-circuiter le tout. De toute manière vous pourrez pas ajouter de site s'il n'y as pas de catégories.

Les catégories

En arrivant sur la page, cliquez sur Gérer les catégories, vous aurez alors ceci :

Vous avez donc deux sections : Ajouter et Modifier une catégorie.

Ajouter une catégorie

Pour ajouter une catégorie, rien de plus simple.

Entrez le nom de la catégorie dans le champ de texte puis cliquer sur le bouton "Ajouter une catégorie" :

Vous pourrez alors voir tout en bas qu'une carte avec le nom de la catégorie est apparu :

Les sites

Maintenant qu'on as une catégorie, ou plusieurs, on peux créer un site. Pour cela, cliquez sur "Gérer les sites".

La page devrait alors ressembler à ceci :

Pour la documentation, j'ai créer préalablement trois catégories : Cat1, Cat2 et Cat3.

Ajouter un site

Vous l'aurez compris, pour ajouter un site, tout se passe dans l'encart "Ajouter un site" :

Plusieurs champs s'offrent alors à vous :

  • Titre de la carte : Ce sera le titre de votre site.

  • URL du site : Le lien pour accéder à votre site.

  • Tag pour le site : Vous pouvez mettre un tag personnalisé.

  • Description du site : Une courte description de votre site.

  • Catégorie : Vous devrez placer votre site dans l'une des catégories que vous avez précédemment créer.

Attention :

Le champ Tag pour le site disparaîtra dans une future version, les tags étant modifié automatiquement par le système afin d'indiquer si un site est accessible ou non.

Une fois que le site sera créer, il va apparaître en bas de la page dans la section "Sites existants" :

En cliquant sur le bouton "Supprimer", vous pouvez bien entendu supprimer un des sites.
