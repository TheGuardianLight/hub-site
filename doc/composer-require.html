Composer require

Une fois que le serveur web ainsi que PHP soit installé et configuré, il reste encore à installer des modules nécessaire au fonctionnement de l'outil.

Dans la racine du projet qui a été clôné, il y a un fichier composer.json qui contient tout les modules nécessaire.

Pour les installer, il faudra exécuter la commande ci-dessous dans le même dossier:

composer require

Vous aurez alors une série de question auquel il faudra simplement appuyer sur la touche entré deux fois, ce qui va ensuite lancer le téléchargement et l'installation des modules nécessaire.

Exemple de fichier composer.json:

{ "require": { "vlucas/phpdotenv": "5.4", "ext-curl": "*", "ext-pdo": "*", "phpmailer/phpmailer": "6.9.1" } }

Une fois que tout les modules seront installé, vous serez alors prêt à installer la base de donnée.
