Base de donnée

Ceci est la dernière étape avant de pouvoir commencer à utiliser l'outil !

Installation du SGBD

SGBD = Système de Gestion de Base de Données

Avant d'installer la base de donnée, il nous faut un SGBD. Cela peut-être PostgreSQL ou MySQL.

Pour l'outil hub-site, il est recommandé d'utiliser MariaDB qui est elle-même basé sur MySQL.

Vous pouvez l'installer avec apt:

apt install mariadb-server

Une fois MariaDB installé, lançé la configuration initiale :

mysql_secure_installation

Puis répondez aux questions suivantes :

Enter current password for root (enter for none)

Ne rien renseigner et appuyer sur la touche "entrer".

Switch to unix_socket authentication

Répondre “n”.

Change the root password?

Répondre “n”.

Remove anonymous users?

Répondre “y”.

Disallow root login remotely?

Répondre "y".

Remove test database and access to it?

Répondre "y".

Reload privilege tables now?

Répondre "y".

MariaDB est maintenant configuré. Il est temps de créer la base de donnée et son utilisateur.

Création de la base de donnée et de l'utilisateur

mysql -p -u root

Puis dans la console, entrez les commande suivantes :

CREATE DATABASE <nom de base de donnée>; CREATE USER '<utilisateur>'@'%' IDENTIFIED BY '<un mot de passe>'; GRANT ALL PRIVILEGES ON <nom de base de donnée>.* TO '<utilisateur>'@'%' WITH GRANT OPTION; FLUSH PRIVILEGES;

Ce script SQL va respectivement faire ceci :

  1. Créer une base de donnée avec le nom que vous avez renseigné.

  2. Créer un utilisateur avec un mot de passe qui lui est associé.

  3. Lui donner tout les privilège sur la base de donnée.

  4. Réactualiser les privilèges pour s'assurer qu'ils soient appliqué correctement.

Installation finale

Pour finaliser l'installation, il ne vous reste plus qu'à vous rendre sur l'adresse https://monsite.fr/install.html ce qui vous affichera une page comme ceci :

Vous aurez alors à remplir les champs suivants :

Nom de la base de données

Indiquer le nom de la base de donnée créer précédement.

Nom d'utilisateur

Nom de l'utilisateur créer précédément.

Mot de passe

Mot de passe de l'utilisateur.

Hôte de la base de données

Adresse IP ou nom d'hôte du serveur hébergant la base de donnée. (Par exemple 192.168.1.2, localhost, bdd.monsite.fr, etc...)

Port

Port sur laquelle votre SGBD écoute.

Une fois ces informations renseigné, cliquez sur "Mettre en place la base de données."

Si tout vas bien, vous serez redirigé vers la page de connexion.
