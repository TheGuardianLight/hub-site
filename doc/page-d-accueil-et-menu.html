Page d'accueil et menu

Page d'accueil

Lorsque vous installerez hubsite pour la première fois, lorsque vous vous connecterez et atterrisez sur la page d'accueil, vous ne verrez rien mis à part ceci :

Cela est tout à fait normal car vous n'avez pas de site ou de catégorie d'enregistré.

Pour commencer à enregistrer des sites ou des catégories, rendez vous dans Sites & catégories.

Menu

Concernant le menu, voici comment il est réparti.

Gestion

Le sous-menu Gestion regroupe les éléments Gestion des Sites/Catégories et Gestion des données.

Gestion des Sites/Catégories :

C'est sur cette page où vous pourrez ajouter et mettre à jour vos catégories et vos sites.

Gestion des données :

Sur cette page, vous aurez la possibilité d'exporter et d'importer des données. Très utile dans le cas d'une migration d'un serveur à un autre.

Compte

Sur cette page vous pourrez modifier votre nom, prénom, adresse email ainsi que votre mot de passe.

Attention cependant, vous ne pouvez pas modifier votre nom d'utilisateur.

A propos

Vous trouverez ici toutes les informations technique nécessaire du site. A savoir :

  • La version actuellement installé.

  • Les version des technologies utilisés (autre que PHP et le serveur web).

  • Le nom du développeur du site.

  • Le lien vers sa page Github pour tout signalement de problèmes ou de suggestions.

Documentation

Vous trouverez la documentation interne du site à cet endroit. (Ce que vous êtes en train de lire).

Notez que la version de la documentation correspond à celle du site.

Bon pas besoin d'expliquer.
